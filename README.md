## 🛠️ Quick Start

1. **Clone & Install**:
   ```bash
   git clone https://github.com/Waqas-Baloch99/Google_flight.git
   cd Google_flight
   npm install
   ```

2. **Environment Setup**:
   ```bash
   cp .env.example .env
   # Add your RapidAPI key to .env
   ```

3. **Development**:
   ```bash
   npm run dev
   ```
